# Image Prompts — Onetime Suite

Ready-to-paste image-generation prompts for every shipped product's landing page and Product Hunt gallery. One file per product: `<brand-lowercase>.md`.

## Usage

1. Open the product's file. Every file starts with the same **shared style block** — prepend it to each prompt so the whole set renders as one visual family (dark UI, single accent per product, soft glow, high contrast, no text unless the prompt specifies it).
2. Each file contains 8 prompts:
   - **Hero image** — 16:9, premium dark-mode concept visual for the landing page header.
   - **Dashboard mockup** — 16:10, stylized screenshot-like render of the main screen with realistic demo data.
   - **OG / social card** — 1200×630, product name + one-line value prop.
   - **PH gallery (5)** — the five gallery shots from that product's `launch-kit/product-hunt.md`.
3. Generate at the stated aspect ratio. Upscale before publishing. Keep the accent hex exact — it is the product's brand color everywhere (landing page, OG card, Whop listing).

## Shared aesthetic

Near-black background `#0B0E14`, one accent color per product (table below), soft neon glow, subtle grid texture, cinematic rim lighting, clean modern sans-serif UI elements, crisp high-contrast render. No text in images unless a prompt explicitly specifies the exact string.
