## 1. Whop-native licensing — SHIPPED across all 101 apps ✅

The big one. Every app now validates purchases directly against Whop (the purchase
*is* the license — no license keys, no key server).

- **Mechanism:** "Sign in with Whop" (OAuth 2.1 + PKCE, loopback redirect
  `http://127.0.0.1:8734/callback`, client `app_B2TMUEvC9aRUNZ`) → access check
  against the app's own `experience_id` using the **customer's token** — no
  company API key ships in any app. Per-app IDs: `license-experience-map.json`
  (private repo), generated from the product↔experience CSV.
- **Desktop apps:** first-run sign-in → device registered (**3-device cap**,
  self-serve deactivate in-app) → cached; daily background re-check; 10-day
  offline grace. Gate = `license-gate.js` at the top of each Electron main.
- **Self-hosted apps:** `npm run activate` once at install → never phones home
  again. `SKIP_LICENSE_CHECK=1` for CI. Multi-tenant/hosted modes never gated.
- **Device registry LIVE:** `https://license.onetimesuite.com` (Coolify app
  `d11ux2wzd5qvsinzef1aale3`, deploy-key auth, SQLite on a **named persistent
  volume — verified to survive redeploys**).
